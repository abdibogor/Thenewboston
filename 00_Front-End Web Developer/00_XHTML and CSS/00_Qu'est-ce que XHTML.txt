XHTML est HTML écrit en XML.

Qu'est-ce que XHTML?

XHTML signifie E X tensible H yper T ext M arkup L anguage

XHTML est presque identique au HTML

XHTML est plus strict que HTML

XHTML est défini en HTML comme une application XML

XHTML est supporté par tous les principaux navigateurs.


Les différences les plus importantes du HTML:

Structure du document
Le DOCTYPE XHTML est obligatoire

L'attribut xmlns dans <html> est obligatoire

<html>, <head>, <title> et <body> sont obligatoires


Éléments XHTML

Les éléments XHTML doivent être correctement imbriqués

Les éléments XHTML doivent toujours être fermés

Les éléments XHTML doivent être en minuscules

Les documents XHTML doivent avoir un élément racine


Attributs XHTML

Les noms d'attributs doivent être en minuscules

Les valeurs d'attribut doivent être indiquées

La minimisation d'attribut est interdite
